# 👤 Sevadaar Volunteer Guide

Welcome to **Sevadaar – Volunteer Coordination System**

This guide explains how a **Volunteer** can use the app to view tasks, update progress, and complete assigned work.

> 🤝 Thank you for being a part of Sevadaar  
> Your contribution helps NGOs manage service efficiently.

---

# 🚀 Getting Started

## 1️⃣ Install & Open App

Open the Sevadaar mobile app on your phone.

You will see:

- Splash Screen
- Login Screen

---

## 🔐 Login / Register

### ✅ If you already have an account

1. Enter email
2. Enter password
3. Tap **Login**

You will be redirected to the **Volunteer Dashboard**

---

### 📝 If you are new

1. Tap **Register**
2. Enter name
3. Enter email
4. Enter password
5. Select role → `Volunteer`
6. Tap **Create Account**

After registration, login normally.

---

# 📊 Volunteer Dashboard

After login you will see:

- Assigned Tasks
- Task Status
- Deadlines
- Profile button

You can only see tasks assigned to you.

You cannot see other volunteers’ tasks.

---

# 📋 Task Workflow

Every task follows this flow:

Created → Assigned → In Progress → Completed → Approved

As a volunteer, you control:

Assigned → In Progress → Completed

Admin controls:

Completed → Approved

---

# 📄 Viewing Assigned Tasks

Steps:

1. Open Dashboard
2. Tap on a task
3. Task Detail Screen will open

You can see:

- Title
- Description
- Deadline
- Current Status
- Assigned by admin

---

# 🚧 Start Working on Task

When a task is assigned, status = Assigned

To start:

1. Open task
2. Tap **Start Task**
3. Status becomes → In Progress

This tells admin you started work.

---

# ✅ Mark Task as Completed

After finishing work:

1. Open task
2. Tap **Mark Completed**
3. Status becomes → Completed

Now admin will review.

You cannot approve your own task.

---

# 🛡 Admin Approval

After you complete a task:

Admin will check it.

If correct → Approved

If changes needed → Admin may update or reassign.

Approved tasks are final.

---

# 👤 Profile Screen

From profile you can:

- View your name
- View email
- Logout

To logout:

Tap Logout → Session ends

Next time you open app → Login required

---

# 🔁 Auto Login

Sevadaar uses Firebase login session.

If you do not logout:

App will open directly to dashboard.

No need to login again.

---

# ⚠ Permissions for Volunteer

You CAN:

✅ View assigned tasks  
✅ Update task status  
✅ Mark task completed  
✅ View your profile  

You CANNOT:

❌ Create tasks  
❌ Assign tasks  
❌ Approve tasks  
❌ View all users  
❌ View all tasks  

Only Admin can do that.

---

# 📱 Tips for Volunteers

✔ Update status when you start work  
✔ Complete tasks before deadline  
✔ Check dashboard daily  
✔ Do not share your login  
✔ Contact admin if task is wrong  

---

# ❓ Common Issues

## Cannot see tasks

Possible reasons:

- Task not assigned yet
- Wrong account logged in

Logout → Login again

---

## Cannot mark completed

Check:

Status must be In Progress

If not, start task first.

---

## App logged out automatically

Login again.

Firebase session may expire.
